**MiMo Code** is a terminal AI programming agent from the Xiaomi MiMo team. In fact, it is a tool that helps automate code development right in the terminal: it can read and write files, execute commands, work with Git and solve complex multistep tasks.
## Key Features
- **Memory system.** The main feature of MiMo Code is the ability to maintain context between steps and sessions. Multi-level memory is used for this (including files like `MEMORY.md `and `checkpoint.md `, as well as the SQLite database), so that the agent does not "forget" the architectural decisions made and the state of the project. This is especially important for long tasks (with hundreds of steps), where ordinary chatbots lose focus due to the overflow of the context window.
- **Compose mode.** Allows you to describe a high-level goal, and the agent himself will plan and complete the entire cycle: design, coding, testing and verification.
- **Max Mode.** To increase reliability, several solutions are generated at each step, after which the best one is selected.
- **Memory maintenance functions.** Commands like `/dream` and `distill` help to clean the history from duplicates and highlight typical workflows for reuse.
- **Voice control.** Speech recognition via MiMo‑ASR is supported (available for authorized users).
## Technical basis and license
MiMo Code is based on OpenCode and released under the MIT license, which means that it can be freely used, modified, and even used in commercial projects. The source code is open and available on GitHub.
## How is it installed
For macOS and Linux, they use the same command in the terminal:
``bash
curl -fsSL https://mimo .xiaomi.com/install | bash
``
Can be installed on Windows via npm:
```bash
npm install -g @mimo-ai/cli

Install the Extension in VS Code
Open VS Code.

Go to Extensions (sidebar icon) or press Cmd+Shift+X.

Search for MiMoCode (author: xucux).

Click Install.

```
---
It is important to make sure that the path to the executable files is added to the variable during installation.

How to set up VS Code
Download and install VS Code from code.visualstudio.com.

Open the editor and optionally customize theme, font, and size via File → Preferences → Settings (or Code → Preferences → Settings on Mac).

Install relevant extensions for your language (Rust, Python, JS, etc.) in the Extensions view (square icon on the left).

Configure project settings using .vscode/settings.json, .vscode/tasks.json, and .vscode/launch.json for build/debug workflows.

About Mimo AI / Mimo Code
There is no official “Mimo AI” VS Code plugin. If you’re referring to Mimo Code (the CLI tool you tried on Mac), it’s not an extension but a command‑line tool.

For CLI usage: ensure the binary is on your PATH so the shell can find it.

For an AI coding assistant in VS Code, consider alternatives like GitHub Copilot, Tabnine, CodeWhisperer, or GigaCode.
