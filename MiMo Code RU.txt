**MiMo Code** — это терминальный AI‑агент для программирования от команды Xiaomi MiMo. По сути, это инструмент, который помогает автоматизировать разработку кода прямо в терминале: он может читать и писать файлы, выполнять команды, работать с Git и решать сложные многошаговые задачи.

## Ключевые особенности

- **Система памяти.** Главная фишка MiMo Code — умение сохранять контекст между шагами и сессиями. Для этого используется многоуровневая память (в том числе файлы вроде `MEMORY.md` и `checkpoint.md`, а также SQLite‑база), чтобы агент не «забывал» принятые архитектурные решения и состояние проекта. Это особенно важно для длинных задач (на сотни шагов), где обычные чат‑боты теряют фокус из‑за переполнения контекстного окна.
- **Режим Compose.** Позволяет описать высокоуровневую цель, а агент сам спланирует и выполнит весь цикл: проектирование, кодирование, тестирование и проверку.
- **Max Mode.** Для повышения надёжности на каждом шаге генерируется несколько вариантов решения, после чего выбирается лучший.
- **Функции обслуживания памяти.** Команды вроде `/dream` и `distill` помогают чистить историю от дублей и выделять типовые рабочие процессы для повторного использования.
- **Голосовое управление.** Есть поддержка распознавания речи через MiMo‑ASR (доступно для авторизованных пользователей).

## Техническая основа и лицензия

MiMo Code построен на базе OpenCode и выпущен под лицензией MIT — это значит, что его можно свободно использовать, модифицировать и даже применять в коммерческих проектах. Исходный код открыт и доступен на GitHub.

## Как устанавливается

Для macOS и Linux используют одну команду в терминале:
```bash
curl -fsSL https://mimo.xiaomi.com/install | bash
```
На Windows можно поставить через npm:
```bash
npm install -g @mimo-ai/cli
```

---
При установке важно убедиться, что путь к исполняемым файлам добавлен в переменную `PATH` — иначе система не увидит команду `mimo` после установки.


Как настроить VS Code
Скачайте и установите VS Code с официального сайта code.visualstudio.com.

Откройте редактор, при желании настройте тему, шрифт и размер текста в настройках (File → Preferences → Settings или Code → Preferences → Settings на Mac).

Установите нужные расширения (например, для Rust, Python, JavaScript) через вкладку Extensions (значок квадратов слева) — в поиске введите название языка или инструмента.

Настройте конфиги, если нужно: файлы settings.json, tasks.json, launch.json в папке .vscode проекта задают поведение редактора, сборку и отладку.

Про Mimo AI / Mimo Code
Тут важно уточнение: отдельного официального плагина «Mimo AI» для VS Code нет. Если ты имеешь в виду Mimo Code (который ты раньше пробовал на Mac), это скорее CLI‑инструмент, а не плагин.

Если нужен именно CLI: проверь, что путь к исполняемому файлу есть в PATH (на Mac это частая причина ошибки zsh: command not found: mimo).

Альтернативы, которые реально существуют как плагины: GitHub Copilot, Tabnine, Amazon CodeWhisperer, GigaCode (если доступен в твоём окружении).
